------------Selectors:-----------

Universal Selectors:

The universal selector is an asterisk; it is like a wildcard and matches all element types in the document.

For example- This web page is styled using universal selector, which means if any other selector is not applicable then universal selector is used for it. Here, say background color.

*{ background-color: dimgrey; }


Type Selectors:

The type selector matches all the elements specified in the comma-delimited list. It allows to apply the same rules to several elements.

For example- All the paragraphs in this web page are styled under the type selector.

p{ color: azure; font-style: italic; }


Class Selectors:

The class selector allows to make a rule with an element carrying a class attribute whose value is specified in the class selector.

For example- All the h2 headers in this web page are styled using class selectors.

In the source code of this web page:
<h2 class="h">Class Selectors:</h2>
<h1 id="i" class="hh">------------Selectors:----------->/h1>

In css file:
.h{ color: darkcyan; background-color: aquamarine; }
.hh{ color: cadetblue; text-align: center; background-color: lime; }
.b{ padding: 0.1% 2% 0.6% 2%; box-shadow: 0 8px 8px 0;color: aqua; }


Id Selectors:

The id selector works just lik a class selector, but works on the value of id attributes. It uses a # sign before an id attribute.

For example- The h1 header and all the bold and underlined texts in this web page are styled using id selector.

In the source code of this web page:
<a id="i">id selector</a>
<a id="i">class selector</a>
<a id="i">universal selector</a>
<a id="i">type selector</a>
<h1 id="i" class="hh">------------Selectors:----------->/h1>
<b id="j">Vineet Kumar Gupta<br>Roll no.- 1705094<br>CS-2</b>

In css file:
#i{ text-decoration: underline; font-weight: bold; font-family: fantasy; }
#j{ text-decoration: underline; font-weight: bold; font-style: italic; font-size: 150%; font-family: cursive;padding-left: 3.6%; padding: 0% 2% 1% 4%; box-shadow: 0 8px 8px 0;color: aqua; }
