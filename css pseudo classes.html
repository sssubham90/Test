CSS Pseudo Classes

What are Pseudo-classes?


A pseudo-class is used to define a special state of an element.
For example, it can be used to:
  • Style an element when a user mouses over it
  • Style visited and unvisited links differently
  • Style an element when it gets focus
A CSS pseudo-class is a keyword added to a selector that specifies a special state of the selected element(s). For example, :hover can be used to change a button's color when the user hovers over it.

Syntax

The syntax of pseudo-classes: selector:pseudo-class { property:value; }

Pseudo Elements


A CSS pseudo-element is used to style specified parts of an element. For example, it can be used to:
  • Style the first letter, or line, of an element
  • Insert content before, or after, the content of an element

Syntax

The syntax of pseudo-elements: selector::pseudo-element { property:value; }

Types of CSS Pseudo Elements:

  • Before
  • After
  • first-line
  • first-letter
  • active and focus
  • hover and visited
